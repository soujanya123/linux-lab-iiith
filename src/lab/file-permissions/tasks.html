1. Create a test folder with few sub-folders and files inside it for practising chmod command.
  2. Change permissions on folder such that while using 'ls' command on folder error 'Permission denied' is obtained instead of contents of the folder.
  3. Change permissions on the folder such that while using 'cd' command to change present working directory to given folder error 'Permission denied' is obtained.
  4. Change permissions on folder such that command 'touch <file_name>' with folder as present working directory results into 'Permission denied' message.
  5. Change permissions on file such that command 'cat <file_name>' results into 'Permission denied' message.
  6. Change permission on file such that its contents can be read with vim but modifying contents leads to 'Permission denied' error message.
  7. Change permission on folder such that cd to it succeeds, but ls fails.
  8. Change permissons on folder such that cd to it succeeds and also 'touch <file_name>' inside folder succeeds but 'ls' on folder results into 'Permission denied' message.
